### 12-5-1
- fork しただけでは、メモリ使用量は増えない。fork 直後の 2 つのプロセスはほとんどの論理アドレスについて元の物理メモリを共有するからである。どちらかのプロセスでメモリへの変更を行うと、その時になって初めて新しい物理メモリが割り当てられる。これを Copy on Write と言う。
